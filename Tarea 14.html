TAREA 14

Búsqueda en Matriz

Descripción

  • Construya un programa en JAVA y RUBY que, al recibir como datos un arreglo bidimensional de NxM elementos de tipo entero y un número entero, determine cuántas veces se encuentra este número dentro del arreglo. Además, que imprima la matriz al final

  • Ejemplo:
    Valor a buscar: 8
    “El número 8 aparece 3 veces en la Matriz: ”
    [+1 21 89 -2]
    [+8 +7 34 +5]
    [+6 -5 +0 +2]
    [21 11 +8 +8]

                import javax.swing.JOptionPane;
                
                public class Tarea14{
                
                    public static void main(String[] args) {
                        int i, n, b, c=0, m, j;
                        int[][] A;
                        String letrero="";
                        String entrada;
                
                        entrada= JOptionPane.showInputDialog("Ingrese el numero de filas: ");
                        n= Integer.parseInt(entrada);
                
                        entrada= JOptionPane.showInputDialog("Ingrese el numero de columnas");
                        m= Integer.parseInt(entrada);
                
                        A = new int[n][m];
                
                        for(i=0; i<=n-1; i++){
                            for(j=0; j<=m-1; j++){
                                entrada= JOptionPane.showInputDialog("Ingrese A["+(i+1)+"]["+(j+1)+"]: ");
                                A[i][j]= Integer.parseInt(entrada);
                            }
                        }
                
                        entrada= JOptionPane.showInputDialog("Ingrese el numero a buscar: ");
                        b= Integer.parseInt(entrada);
                
                        for (i=0; i<=n-1; i++){
                            for (j=0; j<=m-1; j++) {
                                if(A[i][j]==b){
                                    c=c+1;
                                }
                            }
                        }
                
                        for(i=0; i<=n-1; i++){
                            for (j=0; j<=m-1; j++) {
                                letrero= letrero+A[i][j]+" ";
                            }
                            letrero = letrero + "\n     ";
                         }
                
                        letrero="El valor "+b+" aparece "+c+" veces en la Matriz: \nA=" + letrero;
                
                        JOptionPane.showMessageDialog(null, letrero);
                    }
                }
